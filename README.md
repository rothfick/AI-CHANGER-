## ✨ Overview
AI Changer is a cutting-edge web application designed to revolutionize portrait photography through AI-powered transformations. Leverage the advanced capabilities of Google Gemini to apply artistic styles and compelling effects to your images, all from a simple, elegant interface.

## 🧱 Tech Stack
*   **TypeScript**: Ensures robust and scalable client-side application development.
*   **Google Gemini**: Powers the intelligent image and style transformation capabilities.
*   **Vite**: A fast and modern build tool for front-end development.

## 🚀 Features
*   **Image Transformation**: Effortlessly upload and transform portrait images.
*   **Style Prompt Generation**: Utilize AI to interpret and apply desired artistic styles.
*   **Real-time Previews**: Instantly visualize the effect of transformations.

## 🛠️ Quickstart

```bash
npm install
npm run dev
```

## 🗺️ Project Structure
The project is cleanly structured with core application logic residing in the `src` directory, containing TypeScript files for components and utilities. Configuration files like `tsconfig.json` and `vite.config.ts` manage the project's build and compilation settings, while `index.html` serves as the entry point for the web application. Key metadata for the Gemini integration is stored in `metadata.json`.
